# Prinses
The card game by my sister and I 

This is a card game design I made for my sisters birthday. 
The game is bassed on the travel version of Love Letter, which my sister and I used to play all the time. My sister especcialy loved to play to the game. 
To point were my parents got sick of playing the game and the cards of the game got so damaged we couldn't play anymore.

However my sister didn't give up and together we started playing the game with normal playing cards, assigning each card to a playing card. 
Eventually we started bringing in more cards (as a playing deck had more cards than in the original game) and thus our own version of the game was born. 
All of us are terrible at names, and so we quickly started calling it 'prinses' 
(It was only when looking up the orignal game for this project that I realised we never call it by its actual name)

This website is a way to bring the physical cards to live in the digital world :)
